#  MyBudget - Gestionnaire de Budget Étudiant

#  Description
Application web simple et intuitive développée avec **Streamlit** pour aider les étudiants à gérer leurs dépenses quotidiennes et visualiser où va leur argent.

# Fonctionnalités
- Ajout de dépenses par catégorie
- Calcul automatique du total dépensé
- Graphique circulaire interactif (répartition par catégorie)
- Filtrage des dépenses par catégorie
- Suppression des transactions
- Conseils financiers pour étudiants

# Technologies utilisées
- Python 3
- Streamlit
- Pandas
- Plotly
- JSON

# Installation et exécution
```bash
pip install streamlit pandas plotly
streamlit run app.py
